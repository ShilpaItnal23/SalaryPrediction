# SalaryPrediction
Deployment of ML Models in Heroku using Flask Framework

# Steps to Deploy
1. Train the model.
2. Create Web App using Flask
3. Create an account in Heroku(PAAS)
4. Link the github to heroku
5. Deploy the model
